# 单机麻将

# 玩法（红中麻将）
## 基本用牌 总计112张牌：
- 1-9万各4张
- 1-9筒各4张
- 1-9条各4张
- 红中4张

## 核心行牌规则
- 万能红中：红中在手中可以代替任何一张牌去凑成顺子或刻子。
- 不可吃牌：绝对不能吃上家打出的牌。
- 可以碰、杠：可以碰或杠其他玩家打出的牌。
- 红中不能打出/单用：红中通常不能在碰、杠中作为辅助牌，也不能作为单张废牌打出。

## 胡牌与结算规则
- 只能自摸：绝大多数红中麻将都规定不能接炮（不能胡别人打出的牌），必须通过自己摸牌来胡牌。
- 不分大小胡：一般没有普通麻将繁琐的“清一色”、“大三元”等大胡番数，只要符合“四组面子 + 一对将牌”的平胡即可胡牌，所有人胡牌的基础分数都一样。
- 一马当先（扎鸟/抓鸟）：玩家自摸胡牌后，会从牌堆顶部再摸 1 或多个数张牌（称为“扎鸟”），根据摸到牌的数字（如1、5、9代表自己，2、6代表下家等）来决定是否中鸟，中鸟则意味着赢牌的倍数翻倍。

# 实现
- js/sc2d 基于此轻量游戏框架实现
- js/game 游戏业务代码，分以下3层
  1. 游戏逻辑层，模拟服务端行为，目标方便切换至联机模式，可以用server命名
  2. 视图控制层，完全被动承接逻辑层调用、以及将用户操作以回调的方式传递给逻辑层
  3. 视图层，负责游戏画面呈现，初版不使用位图素材，绘制图形+文本来表现麻将牌
